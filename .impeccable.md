## Design Context (REVAMPED)

### Users
Senior technical recruiters and hiring managers at large enterprise tech companies viewing primarily on **mobile during business hours**. They scan 50+ portfolios/week — this site must stop the scroll and make José memorable in seconds. They want proof of skill AND signal of sophistication. Arriving via LinkedIn on phones (70%+ mobile traffic expected).

### Brand Personality
**Sophisticated. Innovative. AI-Forward.**

Not just a technical professional, but one who leverages cutting-edge tools and thinks strategically. Premium quality, refined taste, uses modern methods (including AI) smartly. The interface should feel like visiting a high-end product — polished, intentional, memorable.

### Aesthetic Direction (NEW)
- **Theme**: Light (primary default for mobile readability) + Dark (premium secondary)
- **Reference**: Apple.com, Linear.app — luxury minimalism, refined whitespace, premium typography, subtle motion. Modern tech companies that showcase AI/ML integration.
- **Anti-reference**: Dark portfolios, dark mode first, cluttered dashboards, accordion menus that hide content
- **Visual tone**: Luxury/premium light theme with gold/brass accents. Generous whitespace. Projects displayed as **visual gallery cards** with metrics prominently shown. Signals AI integration throughout.
- **Typography pair**: Premium display font (bold serif or geometric) + refined body (high-quality sans). MUST feel expensive and intentional.
- **Color**: Clean off-white background (`oklch(98% 0.002 255)`), deep charcoal text (`oklch(15% 0.01 255)`), accent color TBD (gold/amber for luxury, or vibrant blue for tech-forward). Possibly incorporate AI tool colors (orange for ChatGPT, etc.)

### Design Principles (REVISED)
1. **Projects are visual galleries.** Not hidden in accordions. Each project shows metrics/stats visually (cards, badges, visual indicators). Recruiters scan without clicking.
2. **Light mode first, premium everywhere.** Clean, spacious, readable on mobile. No dark mode default — that's 2020s thinking. Premium products (Apple, Figma) went light.
3. **Showcase AI integration.** Subtitle or visual marker noting "Built with AI tools" or similar. Signal that José uses modern methods. Could add tool logos (ChatGPT, Claude, etc.) in About or footer.
4. **Whitespace = luxury.** Generous padding, breathing room. Not cramped. Not full of elements. Scarcity signals premium.
5. **Metrics are visible.** 92%, 500K+, 80%, 6h→45min — these are visual anchors. Use charts, large numbers, visual indicators. NOT text-only.
6. **Mobile-first excellence.** Every section optimized for phone viewing. The portfolio is a mobile-first experience that also looks great on desktop.
7. **Accessible & performant.** Fast load, no flash, proper semantic HTML, keyboard nav, high contrast. Premium = reliable.

### Current Layout Structure (as-implemented)

- **Hero:** Full-bleed, Bootstrap container removed. Staggered two-line name (`José Andrés` / `PICADO`). Dark default. Stats bar (92%, 500K+, 350+) below CTAs.
- **About:** Dossier credential table format (`dl/dt/dd`). No prose. One-sentence manifesto above the table.
- **Projects:** Case-study list — 4 full-width accordion rows. Click to expand description. All 4 projects currently confidential (no external links). Project images removed.
- **Skills:** Indexed rows format (already clean — not to be touched).
- **Contact:** Form with validation — academic requirement, minimal styling changes needed.
- **Global:** Transparent navbar (fills on scroll), section numbers 01–04, unified dark background with 1px border separators between sections.
